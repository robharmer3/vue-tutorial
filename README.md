# Vue 3 + Vite
## Simple Counter

### Description
Here is my first attempt with Vue. 
It is a simple heading, button and counter.

The tech that I used for this application was:
- Vue
- Vite
- JavaScript
- CSS
- Node.js

### How to use it?
To view and use the application locally, please follow these instruction:
1) Clone this repo
2) Run "npm install"
3) Run "npm run dev"
